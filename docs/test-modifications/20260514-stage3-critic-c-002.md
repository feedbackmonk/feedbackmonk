### Brief-author typo reconciliation

The handoff brief listed case "401 missing-cookie." This was a likely typo — the submission endpoint is **public** (anon path is intentional) and cookies are not involved at submission time. The correct C2 invariant-1 case is `alg=none` rejection, which is what the test asserts. Documented inline in the test file's module-level doc comment.

### ConnectInfo injection note

Tests inject `ConnectInfo<SocketAddr>` via request extensions because `tower::ServiceExt::oneshot` does not populate it (the production binary uses `into_make_service_with_connect_info`). This is a test-harness detail and is documented in the helper docstring; it does NOT weaken what's being asserted (the IP-bucket extraction still happens, just from a synthetic source).
